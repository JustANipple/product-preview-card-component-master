# Front-end Style Guide

## Layout

The designs were created to the following widths:

- Mobile: 375px
- Desktop: 1440px

## Colors

### Primary

- Dark cyan: hsl(158, 36%, 37%) price + button
- Cream: hsl(30, 38%, 92%) body 

### Neutral

- Very dark blue: hsl(212, 21%, 14%) h1
- Dark grayish blue: hsl(228, 12%, 48%) p
- White: hsl(0, 0%, 100%) main, button paragraph

## Typography

### Body Copy

- Font size (paragraph): 14px

### Font

- Family: [Montserrat](https://fonts.google.com/specimen/Montserrat)
- Weights: 500, 700 Add to cart (700), p (500)

- Family: [Fraunces](https://fonts.google.com/specimen/Fraunces)
- Weights: 700 h1, big price
